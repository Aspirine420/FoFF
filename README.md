# FoFF — умный таймер выключения ПК

**FoFF**  — это небольшая утилита, которая позволяет запланировать автоматическое выполнение действий на вашем компьютере:
- Выключение
- Перезагрузка
- Спящий режим
- Завершение сеанса (блокировка)
- Отключение интернета
- Выключение монитора

## Возможности
- **Два режима таймера**:
  - Обратный отсчёт (указываете количество секунд)
  - Точное время (указываете время, когда нужно выполнить действие)
- **Сворачивание в системный трей** – окно можно закрыть, и программа продолжит работу в фоне (иконка в панели задач).
- **Контекстное меню** в трее: показать окно, скрыть, выйти.
- **Тёмный интерфейс** – приятный для глаз.
- **Компактный размер** – программа не требует установки, запускается из одного EXE-файла.

## Как использовать
1. Скачайте `FoFF.exe` (или распакуйте архив).
2. Запустите файл – откроется главное окно.
3. Выберите действие (например, «Выключить ПК»).
4. Установите таймер:
   - в режиме «Обратный отсчёт» – введите секунды (например, `3600` для 1 часа);
   - в режиме «В заданное время» – укажите время в формате `ЧЧ:ММ` или `ЧЧ:ММ:СС`.
5. Нажмите **Запустить**.
6. Чтобы свернуть программу в трей, нажмите крестик в окне или кнопку «Свернуть в трей».
7. При необходимости остановите таймер кнопкой **Остановить** или завершите работу через меню трея.

## Системные требования
- Windows (7, 8, 10, 11) – основная платформа.
- Linux (поддерживается, но требует дополнительных утилит для некоторых действий).

FoFF — Smart Shutdown Timer
FoFF is a small utility that allows you to schedule automatic actions on your computer:

    Shut down

    Reboot

    Hibernate

    Lock session

    Disconnect Internet

    Turn off monitor

Features

    Two timer modes:

        Countdown (set number of seconds)

        Scheduled time (set exact time when the action should be performed)

    Minimize to system tray – close the window and the program continues running in the background (icon in the taskbar).

    Context menu in tray: show window, hide, exit.

    Dark interface – easy on the eyes.

    Compact size – no installation required, runs from a single EXE file.

How to use

    Download FoFF.exe (or extract the archive).

    Run the file – the main window will open.

    Select an action (e.g., "Shut down PC").

    Set the timer:

        In "Countdown" mode – enter seconds (e.g., 3600 for 1 hour).

        In "Scheduled time" mode – enter time in HH:MM or HH:MM:SS format.

    Click Start.

    To minimize to tray, click the window close button (X) or the "Minimize to tray" button.

    If necessary, stop the timer with the Stop button or exit via the tray menu.

System requirements

    Windows (7, 8, 10, 11) – primary platform.

    Linux (supported but requires additional utilities for some actions).
